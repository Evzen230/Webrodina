# Web Rodina

Web rodina je komunitní platforma navržená pro sdílení rodinných fotografií a uchovávání společných vzpomínek. Původně vznikla jako jednoduché úložiště fotek, ale jejím cílem je vyvinout se v plnohodnotnou rodinnou sociální síť.

---

## 🚀 Jak začít

1. **Adresa:** Web naleznete na [webrodina.evza.cz](https://www.google.com/search?q=https://webrodina.evza.cz).
2. **Přístup:** Pro vstup jsou vyžadovány přihlašovací údaje. Pokud je nemáte, kontaktujte správce (Jan Hrabal).
3. **Používání:** Po přihlášení můžete nahrávat fotografie a využívat všechny dostupné funkce.
---

## 🛠️ Funkce webu

### Aktuální funkce:
* **Fotogalerie:** Nahrávání (i hromadné), organizace do alb a složek, přidávání popisů a štítků.
* **Metadata (EXIF):** Možnost nahrávat a prohlížet technická data o fotkách (datum, zařízení, GPS).
* **Rodinná kronika:** Zápisy důležitých událostí s podporou stylování textu "Parse Down".
* **Interakce:** Komentování fotek a přidávání reakcí (srdíčka).
* **Kalendář:** Evidence narozenin, srazů, oslav a výročí s automatickým upozorněním.
* **Mapa:** Zobrazení polohy pořízení fotek a tvorba heatmapy.
* **Statistiky a profil:** Seznam členů, osobní profily s biografií, odznaky za aktivitu a veřejné statistiky.

### Plánované funkce: 🤫

* Vkládání YouTube videí.
* Počasí pro vybrané lokace.
* Tmavý režim (Dark mode).
* Systém oznámení.
---

## 🏅 Systém odznaků

Uživatelé mohou sbírat odznaky za svou aktivitu na webu.

### **Podle počtu fotek:**
* **Nováček:** < 5 fotek 
* **Profík:** > 100 fotek 
* **Legenda:** > 250 fotek 

### **Speciální odznaky:**

* **Noční pták:** Více než 5 fotek vyfocených mezi 0:00 a 4:00 (vyžaduje metadata).
* **Srdcař:** Více srdíček u vlastních fotek, než je jejich celkový počet.
* **Kronikář:** Více než 10 zápisů do kroniky.
